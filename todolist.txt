* 🚀 30 DAYS BACKEND ROADMAP – FROM ZERO TO SOLID
* =========================
* 🗓️ WEEK 1 – FUNDAMENTAL & STRUKTUR PROJECT
* Fokus: Fondasi Backend yang Kuat
* =========================
* 📅 Hari 1–2: Setup & Struktur ✅
* Install Laravel
* Setup database
* Rapikan folder structure
* Buat model: Product, Category, User
* Pelajari:
* - MVC structure
* - Routing (web.php)
* - Controller basic
* - Blade basic
* 📅 Hari 3–4: Database Design (PENTING BANGET) ✅
* Buat ERD sederhana
* Tabel:
* - users
* - categories
* - products
* - product_images
* - carts
* - cart_items
* Relasi:
* - Category hasMany Product
* - Product belongsTo Category
* - User hasMany Cart
* - Cart hasMany CartItem
* - CartItem belongsTo Product
* Pelajari:
* - Migration
* - Foreign key
* - onDelete cascade
* - Eloquent relationship
* ⚠️ STOP kalau relasi belum paham
* 📅 Hari 5–6: CRUD Product (Tanpa UI Ribet) ✅
* Buat:
* - Create product
* - Edit product
* - Delete product
* - List product
* Gunakan:
* - Form request validation
* - Resource controller
* - Route grouping
* - Mass assignment ($fillable)
* 🎯 Target:
* - Controller bersih
* - Tidak ada logic numpuk

* Pisahkan logic ke ProductService
